# Live Tab Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Live" tab showing today's NBA games with live scores, algo-vs-book line comparison with Edge %, and real-time player stat projections (PTS/REB/AST/3PM/PRA).

**Architecture:** A new API endpoint `/api/live-games.js` fetches ESPN scoreboard + box scores, reads KV conviction/EV plays for algo predictions, then calls BDL season averages for player projection baselines. The frontend adds a "Live" tab with a split layout: game cards (scores + 3-column lines panel: book lines | Edge% | algo lines) on top, player projection table below. Both sections auto-refresh every 30 seconds.

**Tech Stack:** Next.js Pages Router, ESPN public APIs, BallDontLie API v1 (Bearer token), Upstash KV (conviction plays), existing `lib/store.js` helpers, React useState/useEffect, inline styles matching existing `T` theme tokens.

### Projection Formula
```js
const minutesPerPeriod = 12;
const totalMinutes = 48;
const elapsed = (period - 1) * 12 + (12 - clockMinutes); // clockMinutes from displayClock
```

### Edge Computation
```js
// American odds → implied probability
const implied = odds < 0 ? (-odds) / (-odds + 100) : 100 / (odds + 100);

// ML edge: algo win prob vs book implied prob
const mlEdgePct = (ourProb / 100 - bookImplied) * 100;

// Spread edge: algo spread vs book spread (difference in half-points)
const spreadEdgePts = algoSpread - bookSpread; // negative = algo more negative (stronger fav)

// Total edge: algo total vs book total (in points)
const totalEdgePts = algoTotal - bookTotal; // positive = lean OVER
```

### Algo Line Conversions
```js
// ourProbability (0-100) → American moneyline
function probToML(prob) {
  if (prob >= 50) return Math.round(-(prob / (100 - prob)) * 100);
  return Math.round(((100 - prob) / prob) * 100);
}

// Win prob → predicted margin (NBA empirical: ~0.4 pts per 1% above 50%)
function probToSpread(prob) {
  return -((prob - 50) * 0.4 * 2); // negative = home team favored margin
}
```

```js
const BDL_BASE = "https://api.balldontlie.io/v1";

async function bdlFetch(path) {
  const key = process.env.BALLDONTLIE_API_KEY;
  if (!key) return null;
  try {
    const res = await fetch(`${BDL_BASE}${path}`, {
      headers: { Authorization: `Bearer ${key}` },
    });
    if (!res.ok) return null;
    return res.json();
  } catch { return null; }
}

function norm(s) {
  return (s || "").toLowerCase().replace(/[^a-z]/g, " ").replace(/\s+/g, " ").trim();
}

// American odds → implied probability (0–1)
function impliedProb(odds) {
  if (!odds) return 0.5;
  return odds < 0 ? (-odds) / (-odds + 100) : 100 / (odds + 100);
}

// Win probability (0–100) → American moneyline
function probToML(prob) {
  if (prob >= 50) return Math.round(-(prob / (100 - prob)) * 100);
  return Math.round(((100 - prob) / prob) * 100);
}

// Win probability → predicted point spread (home perspective, negative = home favored)
function probToSpread(homeProb) {
  return +(-(homeProb - 50) * 0.4).toFixed(1);
}
```

```js
async function fetchScoreboard() {
  const res = await fetch(
    "https://site.api.espn.com/apis/site/v2/sports/basketball/nba/scoreboard",
    { cache: "no-store" }
  );
  if (!res.ok) return [];
  const d = await res.json();
  return (d.events || []).map(ev => {
    const comp = ev.competitions?.[0];
    const away = comp?.competitors?.find(c => c.homeAway === "away");
    const home = comp?.competitors?.find(c => c.homeAway === "home");
    const st = comp?.status?.type;
    return {
      id: ev.id,
      away: {
        abbr: away?.team?.abbreviation || "",
        name: away?.team?.displayName || "",
        score: away?.score ?? null,
        record: away?.records?.[0]?.summary || "",
        logo: away?.team?.logo || "",
      },
      home: {
        abbr: home?.team?.abbreviation || "",
        name: home?.team?.displayName || "",
        score: home?.score ?? null,
        record: home?.records?.[0]?.summary || "",
        logo: home?.team?.logo || "",
      },
      status: {
        state: st?.state || "pre",       // "pre" | "in" | "post"
        live: st?.state === "in",
        final: st?.completed || false,
        period: comp?.status?.period || 0,
        displayClock: comp?.status?.displayClock || "",
      },
      date: ev.date,
    };
  });
}
```

```js
async function fetchBoxScore(eventId) {
  try {
    const res = await fetch(
      `https://site.api.espn.com/apis/site/v2/sports/basketball/nba/summary?event=${eventId}`,
      { cache: "no-store" }
    );
    if (!res.ok) return [];
    const d = await res.json();
    const players = [];
    for (const teamEntry of (d.boxscore?.players || [])) {
      const teamAbbr = teamEntry.team?.abbreviation || "";
      for (const athlete of (teamEntry.statistics?.[0]?.athletes || [])) {
        const stats = athlete.statistics || [];
        // ESPN box score stat order: MIN FG 3PT FT OREB DREB REB AST STL BLK TO PF PTS
        const [min, , threes, , , , reb, ast, , , , , pts] = stats;
        const parseMin = (m) => {
          if (!m || m === "--" || m === "0:00") return 0;
          const parts = m.split(":");
          return parseInt(parts[0], 10) + (parseInt(parts[1], 10) || 0) / 60;
        };
        const tpmMade = threes ? parseInt((threes || "0").split("-")[0], 10) : 0;
        players.push({
          espnId: athlete.athlete?.id || null,
          name: athlete.athlete?.displayName || "",
          teamAbbr,
          live: {
            pts: parseInt(pts, 10) || 0,
            reb: parseInt(reb, 10) || 0,
            ast: parseInt(ast, 10) || 0,
            tpm: tpmMade,
            min: parseMin(min),
          },
        });
      }
    }
    return players;
  } catch { return []; }
}
```

```js
// Returns: { [normTeamName]: { homeWinProb, algoSpread, algoTotal, allLines } }
async function buildAlgoMap() {
  const [convPlays, evBets] = await Promise.all([
    getConvictionPlays(),
    getCurrentBets(),
  ]);

  const algoMap = {}; // norm(teamName) → data

  for (const play of (convPlays || [])) {
    // selection: "Boston Celtics ML"
    const teamName = (play.selection || "").replace(/ ML$/i, "").trim();
    const key = norm(teamName);
    if (!algoMap[key]) algoMap[key] = {};
    algoMap[key].homeWinProb = play.ourProbability;
    algoMap[key].allLines = play.allLines || {};
  }

  for (const bet of (evBets || [])) {
    if (bet.betType === "Game Total") {
      // selection: "Over 218.5" or "Under 218.5"
      const match = (bet.selection || "").match(/(\d+\.?\d*)/);
      if (match) {
        // Store against both teams — we'll match by game later
        const total = parseFloat(match[1]);
        // We'll attach to away team key as a signal
        const key = `total:${total}`;
        algoMap[key] = { algoTotal: total + (bet.edge || 0) * 100, allTotalLines: bet.allLines };
      }
    }
    if (bet.betType === "Spread") {
      const teamName = (bet.selection || "").replace(/[\s-+]\d.*$/, "").trim();
      const key = norm(teamName);
      const match = (bet.selection || "").match(/([-+]\d+\.?\d*)/);
      if (match && algoMap[key]) {
        algoMap[key].algoSpread = parseFloat(match[1]);
      }
    }
  }

  return algoMap;
}
```

```js
const ESPN_TO_BDL_ABBR = {
  GS: "GSW", NO: "NOP", UTAH: "UTA", SA: "SAS", NY: "NYK", WSH: "WAS",
};

async function fetchSeasonAvgsForPlayers(espnPlayers) {
  // Map ESPN player IDs to BDL player IDs via name matching
  if (!espnPlayers.length || !process.env.BALLDONTLIE_API_KEY) return {};

  // Get team abbrs needed
  const abbrs = [...new Set(espnPlayers.map(p => p.teamAbbr))];
  const bdlTeamsRes = await bdlFetch("/teams?per_page=30");
  if (!bdlTeamsRes?.data) return {};

  const byAbbr = {};
  for (const t of bdlTeamsRes.data) {
    if (t.abbreviation) byAbbr[t.abbreviation.toLowerCase()] = t.id;
  }

  // Get BDL team IDs for ESPN abbrs
  const bdlTeamIds = abbrs
    .map(a => byAbbr[(ESPN_TO_BDL_ABBR[a] || a).toLowerCase()])
    .filter(Boolean);

  // Fetch players for each team
  const bdlPlayersByName = {};
  for (const tid of bdlTeamIds) {
    const r = await bdlFetch(`/players?team_ids[]=${tid}&per_page=100`);
    for (const p of (r?.data || [])) {
      const name = norm(`${p.first_name} ${p.last_name}`);
      bdlPlayersByName[name] = p.id;
    }
  }

  // Match ESPN players to BDL IDs
  const bdlIds = [];
  const espnToBdlId = {};
  for (const ep of espnPlayers) {
    const bdlId = bdlPlayersByName[norm(ep.name)];
    if (bdlId) { bdlIds.push(bdlId); espnToBdlId[ep.name] = bdlId; }
  }

  if (!bdlIds.length) return {};

  // Fetch season averages — one per request, batches of 8
  const season = new Date().getMonth() >= 9
    ? new Date().getFullYear() : new Date().getFullYear() - 1;
  const seasonAvg = {};
  const BATCH = 8;
  for (let i = 0; i < bdlIds.length; i += BATCH) {
    const batch = bdlIds.slice(i, i + BATCH);
    const results = await Promise.all(
      batch.map(id => bdlFetch(`/season_averages?season=${season}&player_ids[]=${id}`))
    );
    for (const r of results) {
      for (const a of (r?.data || [])) seasonAvg[a.player_id] = a;
    }
  }

  // Build name → avg map
  const avgByName = {};
  for (const ep of espnPlayers) {
    const bdlId = espnToBdlId[ep.name];
    if (bdlId && seasonAvg[bdlId]) {
      const a = seasonAvg[bdlId];
      avgByName[ep.name] = {
        pts: parseFloat(a.pts) || 0,
        reb: parseFloat(a.reb) || 0,
        ast: parseFloat(a.ast) || 0,
        tpm: parseFloat(a.fg3m) || 0,
        min: parseFloat(a.min) || 0,
      };
    }
  }
  return avgByName;
}
```

- [ ] **Step 6: Add projection computation helper**

```js
// period: 1-4, displayClock: "4:22" remaining in period
function computeProjection(currentStat, seasonAvg, seasonMin, period, displayClock) {
  if (!seasonMin || seasonMin < 5) return null;
  const clockParts = (displayClock || "0:00").split(":");
  const clockMinutes = parseInt(clockParts[0], 10) + (parseInt(clockParts[1], 10) || 0) / 60;
  const elapsed = (period - 1) * 12 + (12 - clockMinutes);
  const remaining = Math.max(0, 48 - elapsed);
  const rate = seasonAvg / seasonMin;
  return +(currentStat + rate * remaining).toFixed(1);
}
```

- [ ] **Step 7: Write the main handler**

```js
export default async function handler(req, res) {
  res.setHeader("Cache-Control", "s-maxage=25, stale-while-revalidate=30");

  try {
    // 1. Get scoreboard + algo map in parallel
    const [games, algoMap] = await Promise.all([
      fetchScoreboard(),
      buildAlgoMap(),
    ]);

    if (!games.length) {
      return res.status(200).json({ games: [], books: [], lastUpdated: new Date().toISOString() });
    }

    // 2. Fetch box scores for live games only (parallel)
    const boxScoreMap = {}; // eventId → players[]
    await Promise.all(
      games.filter(g => g.status.live).map(async g => {
        boxScoreMap[g.id] = await fetchBoxScore(g.id);
      })
    );

    // 3. Collect all live players for BDL lookup
    const allLivePlayers = Object.values(boxScoreMap).flat();
    const seasonAvgByName = allLivePlayers.length > 0
      ? await fetchSeasonAvgsForPlayers(allLivePlayers)
      : {};

    // 4. Build per-game data
    const bookSet = new Set();
    const result = games.map(game => {
      // ── Lines from conviction plays ──
      const homeKey = norm(game.home.name);
      const awayKey = norm(game.away.name);
      const homeAlgo = algoMap[homeKey] || {};
      const awayAlgo = algoMap[awayKey] || {};
      const algoData = Object.keys(homeAlgo).length ? homeAlgo : awayAlgo;

      // Collect all book keys from allLines
      const lines = {};
      for (const [bk, line] of Object.entries(algoData.allLines || {})) {
        bookSet.add(bk);
        if (!lines[bk]) lines[bk] = {};
        lines[bk].ml = line.odds;
      }

      // Algo predictions
      const rawProb = algoData.homeWinProb;
      // If conviction play is for away team, flip probability
      const isHomeAlgo = Object.keys(homeAlgo).length > 0;
      const homeWinProb = rawProb != null
        ? (isHomeAlgo ? rawProb : 100 - rawProb)
        : null;

      const algo = homeWinProb != null ? {
        homeWinProb,
        predictedHomeML: probToML(homeWinProb),
        predictedAwayML: probToML(100 - homeWinProb),
        predictedSpread: probToSpread(homeWinProb),
        predictedTotal: null, // set below if available
      } : null;

      // Try to attach algo total from EV bets
      // (simplification: look for a total in algoMap that's near the book total)
      for (const [k, v] of Object.entries(algoMap)) {
        if (k.startsWith("total:") && v.algoTotal) {
          if (algo) algo.predictedTotal = +v.algoTotal.toFixed(1);
        }
      }

      // Edge computation (vs first available book line)
      let edge = null;
      const firstBook = Object.keys(lines)[0];
      if (algo && firstBook && lines[firstBook]?.ml) {
        const bookML = lines[firstBook].ml;
        const bookProb = impliedProb(bookML); // for home if bookML < 0
        const mlEdgePct = +((homeWinProb / 100 - bookProb) * 100).toFixed(1);
        edge = {
          ml: {
            pct: mlEdgePct,
            lean: mlEdgePct > 0 ? "home" : mlEdgePct < 0 ? "away" : null,
            label: mlEdgePct > 2 ? `${game.home.abbr} EDGE`
                 : mlEdgePct < -2 ? `${game.away.abbr} EDGE`
                 : "NEUTRAL",
          },
          spread: algo.predictedSpread != null ? {
            pts: algo.predictedSpread,
            label: algo.predictedSpread < -1 ? "HOME COVER"
                 : algo.predictedSpread > 1 ? "AWAY COVER"
                 : "PUSH",
          } : null,
          total: algo.predictedTotal ? {
            pts: null, // computed per-book in frontend when book total is known
            direction: null,
          } : null,
        };
      }

      // ── Players ──
      const livePlayers = boxScoreMap[game.id] || [];
      const players = livePlayers
        .map(ep => {
          const avg = seasonAvgByName[ep.name];
          const proj = avg && game.status.live ? {
            pts: computeProjection(ep.live.pts, avg.pts, avg.min, game.status.period, game.status.displayClock),
            reb: computeProjection(ep.live.reb, avg.reb, avg.min, game.status.period, game.status.displayClock),
            ast: computeProjection(ep.live.ast, avg.ast, avg.min, game.status.period, game.status.displayClock),
            tpm: computeProjection(ep.live.tpm, avg.tpm, avg.min, game.status.period, game.status.displayClock),
          } : null;
          const pra = proj
            ? +((proj.pts || 0) + (proj.reb || 0) + (proj.ast || 0)).toFixed(1)
            : null;
          return {
            name: ep.name,
            espnId: ep.espnId,
            teamAbbr: ep.teamAbbr,
            live: ep.live,
            season: avg || null,
            proj: proj ? { ...proj, pra } : null,
          };
        })
        .filter(p => p.live.min > 0 || p.live.pts > 0)
        .sort((a, b) => (b.proj?.pra ?? b.live.pts) - (a.proj?.pra ?? a.live.pts));

      return { ...game, lines, algo, edge, players };
    });

    const books = [...bookSet];
    return res.status(200).json({
      games: result,
      books,
      lastUpdated: new Date().toISOString(),
    });
  } catch (e) {
    console.error("[LiveGames]", e);
    return res.status(500).json({ error: e.message, games: [] });
  }
}
```

```jsx
                background: active ? `${meta.color}18` : T.surface,
                border: `1px solid ${active ? meta.color : T.border}`,
                color: active ? meta.color : T.textDim,
                cursor:"pointer", transition:"all 0.12s",
                fontFamily:"'Barlow',sans-serif",

        </div>

      {/* ── Loading / empty state ── */}
      {liveLoading && !liveData && (

                fontWeight:400 }}>

              </span>

          </div>

          {games.map(game => {
            const bkLines = activeBook ? game.lines?.[activeBook] : null;
            const algo = game.algo;
            const edge = game.edge;
            const liveGame = game.status.live;
            const finalGame = game.status.final;

            // Compute total edge if we have both algo total and book total
            let totalEdgePts = null;
            if (algo?.predictedTotal && bkLines?.total) {
              totalEdgePts = +(algo.predictedTotal - bkLines.total).toFixed(1);
            }

            return (
              <div key={game.id} style={{
                background: T.surface,
                border: `1px solid ${liveGame ? "#22c55e22" : T.border}`,
                borderRadius:10, marginBottom:12, overflow:"hidden",
              }}>
                {/* Score row */}
                <div style={{ padding:"12px 14px 10px" }}>
                  <div style={{ display:"flex", justifyContent:"space-between",
                    alignItems:"center", marginBottom:10 }}>
                    {liveGame ? (
                      <span style={{ background:"#7f1d1d", color:"#fca5a5",
                        fontSize:9, fontWeight:700, padding:"2px 7px", borderRadius:3,
                        letterSpacing:"0.06em",
                        animation:"livePulse 1.5s ease-in-out infinite" }}>
                        ● LIVE{game.status.period ? ` Q${game.status.period}` : ""}
                        {game.status.displayClock ? ` ${game.status.displayClock}` : ""}
                      </span>
                    ) : finalGame ? (
                      <span style={{ background:T.surfaceHi, color:T.textDim,
                        fontSize:9, fontWeight:700, padding:"2px 7px", borderRadius:3 }}>
                        FINAL
                      </span>
                    ) : (
                      <span style={{ background:"#1a2540", color:"#60a5fa",
                        fontSize:9, fontWeight:700, padding:"2px 7px", borderRadius:3 }}>
                        {game.date ? new Date(game.date).toLocaleTimeString("en-US",
                          { hour:"numeric", minute:"2-digit", timeZoneName:"short" }) : "Upcoming"}
                      </span>
                    )}

                      <div style={{ fontSize:16, fontWeight:800, color:T.text,
                        letterSpacing:"0.04em", fontFamily:"'Barlow',sans-serif" }}>
                        {game.away.abbr}
                      </div>
                      {game.away.score != null && (
                        <div style={{ fontSize:26, fontWeight:900, color:"#fff",
                          fontFamily:"'JetBrains Mono',monospace", lineHeight:1 }}>
                          {game.away.score}
                        </div>
                      )}
                      <div style={{ fontSize:9, color:T.textDeep, marginTop:2 }}>
                        {game.away.record} · Away
                      </div>
                    </div>
                    <div style={{ color:T.border, fontSize:13, padding:"0 12px" }}>@</div>
                    <div style={{ flex:1, textAlign:"right" }}>
                      <div style={{ fontSize:16, fontWeight:800, color:T.text,
                        letterSpacing:"0.04em", fontFamily:"'Barlow',sans-serif" }}>
                        {game.home.abbr}
                      </div>
                      {game.home.score != null && (
                        <div style={{ fontSize:26, fontWeight:900, color:"#fff",
                          fontFamily:"'JetBrains Mono',monospace", lineHeight:1 }}>
                          {game.home.score}
                        </div>
                      )}
                      <div style={{ fontSize:9, color:T.textDeep, marginTop:2 }}>
                        Away · {game.home.record}
                      </div>
                    </div>
                  </div>
                </div>

                        {/* ML */}
                        <div style={{ textAlign:"right" }}>
                          <div style={{ fontSize:9, color:T.textDeep, textTransform:"uppercase",
                            letterSpacing:"0.06em", marginBottom:2 }}>Predicted ML</div>
                          {algo?.predictedHomeML ? (
                            <>
                              <div style={{ fontSize:13, fontWeight:700, color:"#a78bfa",
                                fontFamily:"'JetBrains Mono',monospace" }}>
                                {algo.predictedHomeML > 0 ? `+${algo.predictedHomeML}` : algo.predictedHomeML}

                            </>
                          ) : <div style={{ fontSize:11, color:T.textDeep }}>—</div>}
                        </div>
                        {/* Spread */}
                        {algo?.predictedSpread != null && (
                          <div style={{ textAlign:"right" }}>
                            <div style={{ fontSize:9, color:T.textDeep, textTransform:"uppercase",
                              letterSpacing:"0.06em", marginBottom:2 }}>Predicted Spread</div>
                            <div style={{ fontSize:13, fontWeight:700, color:"#a78bfa",
                              fontFamily:"'JetBrains Mono',monospace" }}>
                              {algo.predictedSpread > 0 ? `+${algo.predictedSpread}` : algo.predictedSpread}

                            <div style={{ fontSize:10, color:T.textDim,
                              fontFamily:"'JetBrains Mono',monospace" }}>
                              {game.home.abbr} avg margin

                          </div>
                        )}
                        {/* Total */}
                        {algo?.predictedTotal && (
                          <div style={{ textAlign:"right" }}>
                            <div style={{ fontSize:9, color:T.textDeep, textTransform:"uppercase",
                              letterSpacing:"0.06em", marginBottom:2 }}>Predicted Total</div>
                            <div style={{ fontSize:13, fontWeight:700, color:"#a78bfa",
                              fontFamily:"'JetBrains Mono',monospace" }}>
                              {algo.predictedTotal}
                            </div>
                            <div style={{ fontSize:10, color:T.textDim,
                              fontFamily:"'JetBrains Mono',monospace" }}>
                              pace-adjusted
                            </div>
                          </div>
                        )}
                      </div>
                    </div>
                  </div>
                )}
              </div>
            );
          })}

          {/* ── Player projections ── */}
          <div style={{ fontSize:9, fontWeight:700, color:T.textDeep, letterSpacing:"0.12em",
            textTransform:"uppercase", display:"flex", alignItems:"center", gap:8,
            marginBottom:10, marginTop:16, fontFamily:"'Barlow',sans-serif" }}>
            Player Projections · Live
            <div style={{ flex:1, height:1, background:T.border }} />
          </div>

          {/* Category filter pills */}
          <div style={{ display:"flex", gap:4, marginBottom:10, flexWrap:"wrap" }}>
            {["All","Points","Rebounds","Assists","3PM","PRA"].map(cat => (
              <button key={cat} onClick={() => setLivePropCat(cat)} style={{
                padding:"4px 10px", borderRadius:4,
                fontSize:10, fontWeight:600, cursor:"pointer",
                background: cat === livePropCat ? `${T.blue}20` : T.surface,
                border: `1px solid ${cat === livePropCat ? T.blue : T.border}`,
                color: cat === livePropCat ? T.blue : T.textDim,
                fontFamily:"'Barlow',sans-serif", transition:"all 0.12s",
              }}>{cat}</button>
            ))}
          </div>

          {sortedPlayers.length === 0 ? (
            <div style={{ textAlign:"center", padding:"24px 0", color:T.textDim, fontSize:12 }}>
              {games.some(g => g.status.live) ? "Loading player stats…" : "Player stats available once games start"}
            </div>
          ) : (
            <div style={{ background:T.surface, border:`1px solid ${T.border}`,
              borderRadius:8, overflow:"hidden" }}>
              {/* Header */}
              <div style={{ display:"grid",
                gridTemplateColumns:"1fr 44px 44px 44px 44px 52px",
                gap:4, padding:"7px 12px",
                background:T.bg, borderBottom:`1px solid ${T.border}` }}>
                {["Player","PTS","REB","AST","3PM"].map(h => (
                  <div key={h} style={{ fontSize:9, fontWeight:700, color:T.textDeep,
                    letterSpacing:"0.08em", textTransform:"uppercase",
                    fontFamily:"'Barlow',sans-serif",
                    textAlign: h === "Player" ? "left" : "right" }}>{h}</div>
                ))}
                <div style={{ fontSize:9, fontWeight:700, color:T.blue,
                  letterSpacing:"0.08em", textTransform:"uppercase",
                  fontFamily:"'Barlow',sans-serif", textAlign:"right" }}>PRA ▾</div>
              </div>

              {sortedPlayers.slice(0, 30).map((p, i) => (
                <div key={`${p.name}-${i}`} style={{
                  display:"grid",
                  gridTemplateColumns:"1fr 44px 44px 44px 44px 52px",
                  gap:4, padding:"8px 12px",
                  borderBottom: i < sortedPlayers.slice(0,30).length - 1
                    ? `1px solid ${T.border}20` : "none",
                  alignItems:"center",
                }}>
                  <div>
                    <div style={{ fontSize:12, color:T.text, fontWeight:600,
                      fontFamily:"'Barlow',sans-serif" }}>{p.name}</div>
                    <div style={{ fontSize:10, color:T.textDim }}>
                      {p.teamAbbr}
                      {p.gameStatus.live && (
                        <span style={{ color:"#22c55e", marginLeft:4, fontSize:9 }}>● LIVE</span>
                      )}
                    </div>
                  </div>
                  {["pts","reb","ast","tpm"].map(stat => (
                    <div key={stat} style={{ textAlign:"right" }}>
                      <div style={{ fontSize:12, color:T.text,
                        fontFamily:"'JetBrains Mono',monospace" }}>
                        {p.live[stat] ?? "—"}
                      </div>
                      {p.proj?.[stat] != null && (
                        <div style={{ fontSize:9, color:"#16a34a",
                          fontFamily:"'JetBrains Mono',monospace" }}>
                          →{p.proj[stat]}
                        </div>
                      )}
                    </div>
                  ))}
                  <div style={{ textAlign:"right", fontSize:13, fontWeight:700,
                    color: T.blue, fontFamily:"'JetBrains Mono',monospace" }}>
                    {p.proj?.pra ?? (p.live.pts + p.live.reb + p.live.ast) || "—"}
                  </div>
                </div>
              ))}
            </div>
          )}
        </>
      )}

      <style>{`
        @keyframes livePulse { 0%,100%{opacity:1} 50%{opacity:0.55} }
      `}</style>
    </div>
  );
};
```

## Notes for Implementer

- The `allLines` on conviction plays only has **ML odds** (not spread/total). Spread and total lines come from `getCurrentBets()` EV bets. If the cron hasn't run today, KV may be empty — the UI gracefully handles missing `algo`/`lines` data by showing `—`.
- ESPN box score stat columns must be parsed positionally — the `labels` array from the API matches: `["MIN","FG","3PT","FT","OREB","DREB","REB","AST","STL","BLK","TO","PF","PTS"]`
- BDL API key env var: `BALLDONTLIE_API_KEY` (already used in `player-projections.js`)
- The live refresh is 30s on the frontend. The `Cache-Control: s-maxage=25` on the API means Vercel CDN caches for 25s — effectively fresh every 30s without hammering origin.
- If no conviction plays in KV (e.g., off-season), the lines panel simply won't render for games that have no algo data. This is fine.
